
## Description

## Requirement

前端（Frontend）：NVM、Yarn、Node.js

## Build Setup (Local)

1. `yarn create react-app firebase-auth-mail`
2. Install Tailwind CSS with Create React App  https://tailwindcss.com/docs/guides/create-react-app
3. `yarn add react-router-dom`
4. `yarn add firebase`
